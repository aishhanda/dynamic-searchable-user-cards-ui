# Dynamic Searchable User Cards UI

A sleek, interactive UI that displays user profile cards which **filter instantly as you type**. Perfect for dashboards, social apps, admin panels, or any interface that requires **dynamic, searchable data display**.

## Features
- **Dynamic Filtering:** Real-time search updates as you type.
- **Interactive Cards:** Hover effects with blurred background overlay.
- **Responsive Design:** Works seamlessly on mobile, tablet, and desktop.
- **Vanilla JS & CSS:** No frameworks, easy to understand and extend.
